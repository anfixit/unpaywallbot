# Модель угроз для бота обхода paywall

## Активы
1. **Токен Telegram-бота** — доступ к функционалу
2. **Аккаунты пользователей** — credentials для hard paywall
3. **Логи запросов** — могут содержать персональные данные
4. **Кеш статей** — интеллектуальная собственность

## Угрозы
| Угроза | Вероятность | Влияние | Меры защиты |
|--------|-------------|---------|-------------|
| Утечка токена бота | Низкая | Высокое | Хранение в .env, SecretStr |
| Компрометация аккаунтов | Средняя | Высокое | Шифрование session cookies |
| DOS через бота | Средняя | Среднее | Rate limiting middleware |
| Юридические претензии | Низкая | Высокое | Дисклеймер, research only |
