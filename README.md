# unpaywallbot
Бот для авто анализа уязвимостей paywall. Написан на Python (aiogram + Playwright). Реализует 4 метода обхода: отключение JavaScript, подмена Googlebot, headless-авторизация и relay через archive.ph. Архитектура с middleware для аудита, Redis-кешированием и конфигурацией через YAML. Только для образовательных целей и авторизованного тестирования.
